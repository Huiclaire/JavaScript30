## Lesson learned

- `Console` tricks
  - `%s`: string
  - `%d`: integer
  - `%f`: float
  - `%o`: object
  - `%c`: for styled
- Defaults of `console.log`
  - `console.warn`
  - `console.error`
  - `console.info`
- Testing with `assert()`
- Viewing DOM elements
  - `document.querySelector()` an element first
  - `console.log(p)`: log out only a DOM tag of the element
  - `console.dir()`: view the properties of the element
- Grouping together
  - `group()`
  - `groupCollapsed`
  - `groupEnd()`
- `console.count()`
- `console.clear()`
